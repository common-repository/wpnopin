Protect your content and block pinterest.com

== Description ==

wpNoPin adds the meta tag in order to block http://www.pinterest.com

You can switch this on and off for the entire site.

If you have it switched off you can block single pages or posts from pinterest.com

== Installation ==

1. Upload `index.php` to the `/wp-content/plugins/wpNoPin` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
* or
1. Search for wpnopin via Plugins - Add New in the admin area
2. Click on Install
3. Activate plugin

== Frequently Asked Questions ==


= Why do I need this plugin? =
If you have content that you don't want to appear on pinterest.com this plugin lets you block pinterest from accessing your content.
